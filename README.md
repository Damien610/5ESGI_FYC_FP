# 🎓 Workshop MCP : Agents IA pour Résumé et Extraction de Mots-Clés

## 🎯 Objectif du Workshop

Ce workshop vous guide dans la création d'un **Model Context Protocol (MCP)** avec deux agents spécialisés :
- 🤖 **Agent Résumé** : Génère des résumés automatiques de contenu
- 🔍 **Agent Mots-Clés** : Extrait les mots-clés pertinents d'un texte

## 📚 Structure du Workshop

Le workshop est organisé en étapes progressives, chacune avec sa correction vidéo :

### Étape 1 : Configuration de l'environnement [Étape 1](step_1_ollama_configuration.md)
- Installation d'Ollama
- Configuration des modèles IA
- Setup du projet MCP
- 🎥 **Vidéo correction** : `corrections/etape1-setup.mp4`

### Étape 2 : Création de l'agent Résumé
- Implémentation du serveur MCP
- Configuration de l'agent de résumé
- Tests et validation
- 🎥 **Vidéo correction** : `corrections/etape2-agent-resume.mp4`

### Étape 3 : Développement de l'agent Mots-Clés
- Ajout du second agent
- Extraction intelligente de mots-clés
- Intégration au MCP existant
- 🎥 **Vidéo correction** : `corrections/etape3-agent-mots-cles.mp4`

### Étape 4 : Intégration et optimisation
- Communication entre agents
- Interface utilisateur
- Optimisations de performance
- 🎥 **Vidéo correction** : `corrections/etape4-integration.mp4`

## 🚀 Démarrage Rapide

1. Clonez ce repository
2. Suivez les instructions de l'Étape 1
3. Regardez la vidéo de correction si vous êtes bloqué
4. Passez à l'étape suivante
